Problem 1

A soft drink bottler is interested in studying the effects on a filling process. A factorial experiment is run using three factors: percent carbonation (in %), operating pressure (in psi), and line speed (in bpm). The response variable is deviation from the target fill height. The experiment consists of \(n=2\) runs for each of the \(a b c = 12\) treatment combinations. The data is available on Blackboard as an Excel File.

Preliminary analysis

The three-factor factorial experiment with \(a=3\) levels of the carbonation factor (in %), denoted by \(A\), \(b=2\) levels of the operating pressure (in psi), denoted by \(B\), and \(c=2\) levels of the line speed (in bpm), denoted by \(C\). Each treatment combination has \(n=2\) replicates for a total of \(N=a b c n=24\) observations. The response variable \(y\) is deviation from the target fill height.

We describe the observations in the three-factor factorial experiment by the fixed-effects interaction model where treatments are defined to be deviations from the overall mean, \[Y_{i j k \ell} = \mu + \tau_i + \beta_j + \gamma_k + (\tau\!\beta)_{i j} + (\tau\!\gamma)_{i k} + (\beta\!\gamma)_{j k} + (\tau\!\beta\gamma)_{i j k} + \epsilon_{i j k \ell} \begin{cases} i = 1,\ldots,a=3\\ j = 1,\ldots,b=2\\ k = 1,\ldots,c=2\\ \ell = 1,\ldots,n=2\\ \end{cases}\] where \(\mu\) is the overall main effect, \(\tau_i\) is the effect of the \(i\)-th level of factor \(A\), \(\beta_j\) is the effect of the \(j\)-th level of factor \(B\), and \(\gamma_k\) is the effect of the \(k\)-th level of factor \(C\). The interaction effects may also be observed using the same labeling, e.g., \((\tau\!\gamma)_{i k}\) is the interaction effect of the \(i\)-th level of factor \(A\) and the \(k\)-th level of factor \(C\).

(b)

If an interaction plot is parallel, or nearly so, then there is no need to include the interaction term.

In the \(3\)-factor ANOVA model, a \(3\)-factor interaction occurs when a \(2\)-factor interaction depends on the level of the \(3\)rd factor.

(c)

From the earlier results, we saw that factors \(A\) (carbon), \(B\) (pressure), and \(C\) (speed) were statistically significant, but only the interaction between factor \(A\) and \(B\) (carbon and pressure) had any significance.

Additive model

The additive model is given by \[Y_{i j k \ell}^{(R)} = \mu + \tau_i + \beta_j + \gamma_k + \epsilon_{i j k \ell}.\] Thus, \[E(Y_{i j k}^{(R)}) = \mu + \tau_i + \beta_j + \gamma_k.\]

We fit the additive model with:

mod.R = aov(y ~ A+B+C)

We define the estimated means for the models with:

fits.R = predict(mod.R)

The interaction plots for the additive where the left and right plots are respectively for factor \(C\) at levels \(1\) and \(2\).

par(mfrow=c(1,2)) for (i in 1:nlevels(C)) { interaction.plot(A[C==levels(C)[i]],B[C==levels(C)[i]], fits.R[C==levels(C)[i]], xlab="A", ylab="mean response", trace.label="B") }

Interaction model

The model that includes interactions between factors \(A\) (carbon) and \(B\) (pressure) is given by \[Y_{i j k \ell}^{(F)} = \mu + \tau_i + \beta_j + \gamma_k + (\tau\!\beta)_{i j} + \epsilon_{i j k \ell}.\] Thus, \[E(Y_{i j k}^{(F)}) = \mu + \tau_i + \beta_j + \gamma_k + (\tau\!\beta)_{i j}.\]

We fit the interaction model with:

mod.F = aov(y ~ A+B+C+A:B)

We define the estimated means for the models with:

fits.F = predict(mod.F)

The interaction plots for the interaction model where the left and right plots are respectively for factor \(C\) at levels \(1\) and \(2\).

par(mfrow=c(1,2)) for (i in 1:nlevels(C)) { interaction.plot(A[C==levels(C)[i]],B[C==levels(C)[i]], fits.F[C==levels(C)[i]], xlab="A", ylab="mean response", trace.label="B") }

Decision

The interaction model fits show only a slight difference difference from the additive model. As a general rule, we prefer simpler models to complex models, and so we decide to leave out the interaction term.

(d)

par(mfrow=c(1,3)) plot(fits.R ~ A) plot(fits.R ~ B) plot(fits.R ~ C)

We see that each factor has a positive effect on fill height. The factor with the biggest marginal effect on fill height is the percent of carbonation (factor \(A\)).

(iii)

A.C = interaction(A,C) interaction.plot(B,A.C, fits.r)

Maximum setting

In this characterization experiment, we find that the best combination of factor levels is given by a low cutting speed (\(A^-\)), a high tool geometry (\(B^+\)), and a high cutting angle (\(C^+\)).

Explanation

First, we observe that \(A^-\) and \(C^+\) has the best interaction effect (its corresponding line is above all of the others). Second, we see that \(B\) has a positive effect, i.e., as we go from \(B^-\) to \(B^+\), the response increases, and thus \(B^+\) is better. Taking these two facts together, we see that the best combination in this experiment is \((A^-,B^+,C^+)\).

We may also just look at the highest point on the graph and see that \((A^-,B^+,C^+)\) has the highest mean response. However, the interaction plot reveals more, particularly if in follow-up experiments where we use what we learned in this experiment to narrow down our search for an optimal combination (where optimality may be with respect to, say, mean response or minimal variability in the response).

Thus, the estimated mean response \(\hat{Y}_{i j k}^{(R)}\) is given by \[\begin{aligned} \hat{Y}_{i j k}^{(R)} = 3.125 - &I(i=1) 3.625 - I(i=2) 6.25 + I(i=3) 4.250\\ - &I(j=1) 1.375 + I(j=2) 1.375\\ - &I(k=1) 9.58\bar{3} + I(k=2) 9.58\bar{3}. \end{aligned}\]

In R code, this may be brute-force implemented as:

yhat.R = function(i,j,k) { y = 3.125 if (i == 1) y = y - 3.625; if (i == 2) y = y - .625; if (i == 3) y = y + 4.25 if (j == 1) y = y - 1.375; if (j == 2) y = y + 1.375 if (k == 1) y = y - .95833333; if (k == 2) y = y + .95833333 y }

We print out all of the estimates of the mean response with:

for (i in 1:3) for (j in 1:2) for (k in 1:2) cat("Y(",c(i,j,k),") = ",yhat.R(i,j,k),"\n")
