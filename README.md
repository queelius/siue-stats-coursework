# SIUe Statistics Coursework (2021)

Source archive for graduate statistics coursework completed at Southern Illinois University Edwardsville during 2021. The rendered content lives at <https://metafunctor.com/probsets/>.

> **Status:** archived. This repository preserves source `.Rmd` files, data, and rendered `.pdf` artifacts. The migration to <https://metafunctor.com/probsets/> was a one-time event; tooling has been removed.

## Courses

| Code | Course | Semester | Instructor | Live |
|------|--------|----------|------------|------|
| STAT 478 | Time Series Analysis | Spring 2021 | Dr. Beidi | [/probsets/stat478/](https://metafunctor.com/probsets/stat478/) |
| STAT 482 | Regression Analysis | Fall 2021 | Dr. Andrew Neath | [/probsets/stat482/](https://metafunctor.com/probsets/stat482/) |
| STAT 575 | Computational Statistics | Summer 2021 | Dr. Beidi | [/probsets/stat575/](https://metafunctor.com/probsets/stat575/) |
| STAT 579 | Discrete Multivariate Analysis | Spring 2021 | Dr. Beidi | [/probsets/stat579/](https://metafunctor.com/probsets/stat579/) |
| STAT 581 | Statistical Methods | Fall 2021 | Dr. Andrew Neath | [/probsets/stat581/](https://metafunctor.com/probsets/stat581/) |

Each course directory under `stats/` contains assignment subdirectories (`hw1/`, `exam1/`, `project/`, etc.) with `.Rmd` source, data files, and rendered `.pdf` artifacts.
